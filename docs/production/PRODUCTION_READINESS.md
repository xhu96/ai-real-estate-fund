# Production Readiness

AI Real Estate Fund includes an operations layer around the investment committee so the project can be deployed and operated like a real service. The production-readiness scope is software-operational readiness: configuration validation, deployment, auth, rate limiting, audit logs, health checks, observability, runbooks, and release gates.

It does **not** make the model legally or professionally sufficient for real estate investment decisions. Real deployments still require licensed data, professional legal/tax/lending review, security review, privacy review, and model-risk governance.

## Production gate

Run this before every release:

```bash
APP_ENV=production python -m ai_real_estate_fund readiness --strict
```

In CI, use:

```bash
APP_ENV=ci python -m ai_real_estate_fund readiness --strict
```

The production gate checks configuration, required docs, Docker files (`Dockerfile`, `docker-compose.yml`, `compose.production.yml`), CI workflows, deployment scripts, test coverage surface, importability, writable data paths, health checks, security flags, CORS, API keys, payload limits, audit storage, and package metadata placeholders.

## Minimum production controls

A deployable environment should set:

```bash
APP_ENV=production
SECRET_KEY=<32+ random bytes>
REQUIRE_API_KEY=true
API_KEY_HASHES=<sha256 hashes generated by scripts/rotate_api_key.py>
REQUIRE_TLS=true
ENABLE_DEMO_MODE=false
ENABLE_FIXTURE_DATA=false
CORS_ORIGINS=https://app.ai-real-estate-fund.example
ALLOWED_HOSTS=api.ai-real-estate-fund.example
RATE_LIMIT_PER_MINUTE=120
MAX_PAYLOAD_BYTES=1000000
REQUIRE_HUMAN_APPROVAL_FOR_BUY=true
```

Use `.env.production.example` as the starting point.

## Release criteria

A release should not be promoted unless all of the following pass:

```bash
python -m compileall -q src app/backend tests
python -m pytest -q
python scripts/smoke_test.py
APP_ENV=ci python -m ai_real_estate_fund readiness --strict
python scripts/release_manifest.py
```

For a public GitHub release, also replace `xhu96` and `Xhulio L.` placeholders in `pyproject.toml`, `README.md`, and `LICENSE`.
