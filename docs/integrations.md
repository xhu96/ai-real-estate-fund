# Integration Contracts

The integration package contains offline adapter stubs for assessor,
recorder, rent listings, sale listings, property management, accounting,
lending, insurance, crime, schools, census, employment, permits, climate,
flood maps, zoning, title, inspections, contractor bids, appraisals, broker
opinions, bank statements, document storage, geocoding, and AVMs.

They intentionally return deterministic fixture records. A real deployment
should replace each adapter with licensed provider access, retries, rate
limits, provenance, caching, privacy controls, and data-use compliance.
